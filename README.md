# US Bikeshare Data Explorer

**Date created:** 27-Nov-2025

## Project Title
US Bikeshare Data Explorer

## Description
This project analyzes bikeshare usage data from three major US cities: Chicago, New York City, and Washington. The Python script allows users to filter data by city, month, and day of the week and provides insights on the most frequent travel times, popular stations, trip durations, and user demographics. 

## Files Used
- `bikeshare.py` Python script used to analyze the data
- `chicago.csv`, `new_york_city.csv`, `washington.csv` CSV data files (ignored in GitHub via `.gitignore`)
- `README.md` Project documentation
